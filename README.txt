DETALII DE IMPLEMENTARE TEMA 1 METODE NUMERICE - ION VLAD-VASILE, 314CC

castica este construita asemenea indicatiilor din documentatie.

rizare QR am luat ecuatiile din PDF pentru a determina inversa. In comentariile de pe codul celui de-al
doilea task este explicata ecuatia algebrica prin care gasesc vectorul R.

La al treilea task aflu coeficientii si testez cazurile <val1 si >val2.

La final, in PageRank fac din nou citirile necesare si prelucrarile, cu ajutorul functiilor anterior
descrise. Sortarea este facuta cu funtia sortrows, din Octave.
